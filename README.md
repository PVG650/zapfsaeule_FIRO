# Kalibrierung und technische Daten

## Umrechnungen

### Zapfsäuleneingang → Energie

Eine Umdrehung am Zapfsäuleneingang entspricht:

- **0,05 kWh**
- Das Komma befindet sich an der zweiten Stelle der Anzeige.

### Schrittmotor → Zapfsäuleneingang

- 1 Umdrehung des Schrittmotors = **3200 Schritte**
- Das Getriebe hat eine Übersetzung von **14:1**

Daraus ergibt sich:

```text
3200 Schritte × 14 = 44.800 Schritte
```

**44.800 Schritte entsprechen einer Umdrehung am Zapfsäuleneingang.**

## Leistungskalibrierung

Beispiel:

- **220 Schritte/s** entsprechen einer Leistung von **11 kW**

## Drehrichtung

Für ein normales Hochzählen der Anzeige muss die Drehrichtung **negativ** sein.

## Rücksetzen der Anzeige

Zum manuellen Zurücksetzen:

1. Das große seitliche Rad im **Uhrzeigersinn** drehen.
2. Weiterdrehen, bis der vordere Taster im Tal (Vertiefung) steht.
3. Danach befindet sich die Anzeige in der Referenzposition.

## Wichtige Kennwerte

| Parameter | Wert |
|------------|------|
| Schritte pro Motorumdrehung | 3.200 |
| Getriebeübersetzung | 14:1 |
| Schritte pro Umdrehung am Zapfsäuleneingang | 44.800 |
| Energie pro Umdrehung am Zapfsäuleneingang | 0,05 kWh |
| Geschwindigkeit für 11 kW | 220 Schritte/s |
| Zählrichtung | Negativ |
